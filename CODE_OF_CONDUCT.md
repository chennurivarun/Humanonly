# Code of Conduct

We are committed to a respectful, safe, and inclusive community.

## Expected Behavior
- Be constructive and respectful
- Critique ideas, not people
- Assume good intent, ask for clarification

## Unacceptable Behavior
- Harassment or intimidation
- Doxxing or privacy violations
- Deceptive contributions or malicious code

Report concerns to maintainers via private issue or email.
